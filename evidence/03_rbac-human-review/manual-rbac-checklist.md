# Manual UI evidence checklist

Use seeded Manager and BOD accounts; do not mark a step complete without a real screenshot.

- [ ] Manager generates and sees a DRAFT report.
- [ ] Manager sees Evidence and Quality status.
- [ ] Manager approves a valid report; status changes to FINAL.
- [ ] Manager approval is blocked for BLOCKED evidence.
- [ ] BOD report list excludes drafts.
- [ ] BOD sees the finalized report and download buttons.
- [ ] BOD Q&A uses the selected finalized report and exposes no raw employee ID.

Automated domain/API equivalents are recorded in `rbac-results.json`.
